## 現状
- **X(Twitter) → mixi2 への「褒める」機能移行が完了**（実環境でエンドツーエンド動作確認済み：
  返信＋スタンプ付与まで成功）。mixi2 が現在の稼働対象。
- X 版コードはレガシーとして残置（削除せず分岐から外しただけ）。デプロイ対象ではない。
- 実行経路: コミュニティ巡回型。cron で `hometamon_mixi2.py` を定期実行 → 差分取得 →
  分類 → 返信＋スタンプ → DM レポート。
- テスト: `cd main && uv run pytest`（X版＋mixi2版で計83件 green）。
